הוראות המשחק

1. המשחק מורכב מכרטיסים הפוכים עם סמלים שונים.

2. בכל תור, עליך להפוך שני כרטיסים.

3. אם הכרטיסים תואמים, הם נשארים גלויים.

4. אם אינם תואמים, הם יתהפכו חזרה לאחר שנייה.

5. המטרה היא למצוא את כל הזוגות בזמן הקצר ביותר!
